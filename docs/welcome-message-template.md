## Personalization Guide

When using this template:

1. **Fill in all `[bracketed]` fields** with the appropriate information
2. **Personalize the background section** based on what they shared:
   - If they're new to ML: "Since you mentioned you're new to this subject, you're in the right place. This project is designed for learning, and everyone starts somewhere."
   - If they have specific skills: "Your experience with [skill] will be really valuable for our [team name] team."
   - If unsure: "Your fresh perspective is really valuable, and we're excited to have diverse viewpoints on the team."
3. **Adjust the "Ways to Get Involved" section** if they expressed interest in specific areas
4. **Keep the tone encouraging** - emphasize that all skill levels are needed

## Sending Options

- **Email**: Use for formal welcome with the full template
- **Slack/Discord**: Can use a shorter version, linking to the onboarding guide
- **ProjectMap**: If the platform supports direct messaging, adapt for that context
- **In-person**: Use the key points verbally, then email the guide link

## Follow-up Schedule

Consider these gentle check-ins:

- **Day 2-3**: "How was the onboarding guide? Any questions I can answer?"
- **Day 7**: "Have you had a chance to look at the task board? Anything catch your interest?"
- **Day 14**: "How's it going? Found a team or task you'd like to try?"
