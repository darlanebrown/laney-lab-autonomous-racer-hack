# It provides 2-3x faster model inference than onnxruntime by using Intel-specific
# optimizations: MKL-DNN kernels, graph-level layer fusion (Conv+BN+ReLU merged
# into single ops), cache-friendly memory layouts, and vectorized SIMD instructions.
# This is critical for real-time obstacle detection (MiDaS depth) and trained
# steering models running at 10+ FPS on the low-power Atom processor.
# OpenVINO reads .onnx files directly -- no model conversion step needed.
